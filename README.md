# support_bot
Мини бот поддержки для телеграмма, ставиться к основному боту у которого имеется ссылка на поддержку.

Установка пакета для создания окружений (если его нет)
  apt update && apt install python3-venv -y

Создание отдельной папки и файла:
  mkdir /root/support-bot && cd /root/support-bot
Открыть файл и вставить код с заполнением переменных
  nano support.py
